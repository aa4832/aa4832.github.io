How it works: A 24 GHz mmWave radar (Waveshare HMMD / S3KM1110) measures the distance to the nearest moving object in front of my door. Raw sensor readings within a 5-second window are grouped into a single “detection” — one per door-use event — recorded with the closest distance observed. The chart shows every detection for the current day; at midnight ET, the day’s detections are saved to a .txt log file on the sensor.

Why: (1) How often do I actually use my door? (2) When do I use it? (3) Did it open at a time I wasn’t there?
